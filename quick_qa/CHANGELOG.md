# 更新日志

## 1.2.2

- 兼容历史配置：URL 题库整理的 AI 超时低于 300 秒时自动按 600 秒执行，避免旧的 60/90 秒配置继续导致慢速模型提前超时。
- 配置 schema 将 AI 请求超时最小值调整为 300 秒，并提示建议至少 600 秒。

## 1.2.1

- 将 URL 题库整理的默认 AI 超时调整为 600 秒，并允许在配置中最高设置到 1800 秒，适配长网页和慢速推理模型。
- 生成题库时写入关键进度日志：URL 校验、网页抓取、正文清洗、AI 调用、JSON 解析和草稿生成，便于在 TelePilot 配置动作窗口与日志页排查。

## 1.2.0

- 改用 TelePilot 通用配置框架声明题库列表：支持多组题库行展示、编辑、删除、启停和拖拽排序。
- 新增配置页动作 `generate_knowledge_base`，管理员填入 URL 后可直接调用插件受控 HTTP 与 AI 生成题库，并回填到结构化题库列表。
- 运行时兼容旧版 `knowledge_bases_json` 与命令导入题库，同时优先读取新版 `knowledge_bases` 配置；停用的题库不会进入开局选择池。

## 1.1.0

- 题库管理主路径调整为 TelePilot Web 配置页：管理员可在配置里填入 URL，点击“获取并整理为题库”后由平台 AI 生成三选一题库，并保存到账号配置。
- 更新插件使用说明、配置说明和命令帮助，减少对 `quickqa kb import` 命令的依赖；命令导入能力保留为兼容入口。
- 优化网页正文清洗，将 HTML 不可断行空格归一为普通空格，提升 URL 题库整理稳定性。

## 1.0.1

- 修正交互入口 `dispatch_modes` 声明，只保留 TelePilot 当前支持的 `public_keyword`；付款确认继续通过 `events` 与 `event_subscriptions` 接收，避免插件中心规范警告。

## 1.0.0

- 新增快问快答积分淘汰赛插件，支持转账报名、随机玩家选择题库、三选一按钮抢答、答错扣分出局和平台结算公告。
- 新增 URL + AI 题库生成流程：抓取公开网页正文，使用内置系统提示词整理三选一题库草稿，确认后保存到插件本地题库。
- 支持保存多个题库，并在每局开局时通过按钮选择一个或多个题库同时出题。
