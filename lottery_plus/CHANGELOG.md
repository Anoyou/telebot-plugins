# 更新日志
## 1.0.3 (2026-05-21)
- 修复交互 Bot 启动彩票模块后只重复“发送命令”提示的问题；现在正常用户转账只会按金额判定为下注，不再进入短会话式参与流程。
- 交互 Bot 玩法提示改为说明转账金额编码：`基础下注金额 × 注数 + 号码`。
- 彩票帮助与交互查询提示改为使用 `{prefix}` / 当前 userbot 全局指令前缀，不再硬编码逗号。
- 开奖周期改回每日固定时间模型，默认每日 21:00 开奖、开奖前 1 分钟封盘；查询仍使用彩票模块自己的命令入口。

## 1.0.2 (2026-05-21)
- 兼容最新开发文档：在 `plugin.json` 中补齐 `config_schema`，修复配置页不显示配置项的问题。
- 增加 `min_telepilot_version` 字段，并保留 `min_telebot_version` 作为兼容别名。
- 同步版本号到 `manifest.py` 与 `plugin.json`。

## 1.0.1 (2026-05-20)
- 补充交互 Bot 运行入口：实现 `on_interaction(ctx, entry_key, payload)` 最小 hook。
- 保持原有指令与消息监听主逻辑不变，仅新增交互入口声明对应的标准动作返回。
- 同步更新版本号与远程元数据一致性。

## 1.0.0
- 初始版本：将参考 `lottery_plugin.py` 迁移为 TelePilot 远程扩展。
- 支持群内下注、开奖、奖池滚存、历史查询、我的注单、热门号码与统计。
- 提供管理员指令（强制开奖、清盘、赞助增减奖池、退款）。
- 提供完整可配置项（价格、费率、限购、开奖周期、命令别名、消息模板等）。
- 增加配置页只读消息预览，便于上线前校对文案。
