# 更新日志

## 1.1.5 - 2026-05-18

- 修复领取监听在 Telethon sender 未预载时直接跳过的问题，主动解析发送者并保留 sender_id 兜底。
- 文字红包改为 Markdown 代码样式优先，避免当前编辑链路不解析 `<code>` 时原样显示标签。

## 1.1.4 - 2026-05-18

- 文字红包发送增加 HTML、Markdown、纯文本三级兜底，避免富文本解析异常时出现“触发成功但无响应”。

## 1.1.3 - 2026-05-18

- 玩法说明改用 `{prefix}` / `{command}` 预览占位符，配置页示例会跟随系统命令前缀和当前指令名实时渲染。

## 1.1.2 - 2026-05-18

- 修复文字红包口令和结算榜单未按 Telegram HTML 渲染的问题，`<code>` 与可折叠引用不再原样显示。

## 1.1.1 - 2026-05-18

- 配置页新增只读玩法说明，直接展示红包、图片红包、管理命令和领取方式。
- 移除配置区里的“是否启用”字段，统一使用页面上方的账号级“功能总开关”。

## 1.1.0 - 2026-05-18

- 按 TelePilot 0.15 远程模块规范重构入口，改为原生 `Plugin` 生命周期、命令注册和消息监听。
- 配置页模式更新为 `single`，并同步声明 `send_file`、`delete_message` 等实际运行权限。
- 模块运行状态按账号隔离保存，避免不同账号共享红包配置和领取状态。
- 移除导入阶段自动写默认配置文件的副作用，安装校验阶段保持只读。

## 1.0.1 - 2026-05-17

- 完善插件市场描述，明确红包、数学题红包、领取结算和按钮确认能力。
- 配套补充 BYRBQ 插件手工测试清单与静态检查脚本，便于发布前验证。
