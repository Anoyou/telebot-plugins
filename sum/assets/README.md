This directory contains the bundled regular Chinese font used by word cloud rendering.

Bundled font:

- `font.otf`: Noto Sans CJK SC Regular
- License: SIL Open Font License 1.1, see `FONT_LICENSE.txt`

Recommended file names:

- `font.ttc`
- `font.otf`
- `font.ttf`

For VPS/Docker, the bundled `font.otf` is used first. You can replace it with another open-source CJK font such as Noto Sans CJK or Source Han Sans. Do not commit proprietary system fonts.
